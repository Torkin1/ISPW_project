Home

Home page of iNeed, which sums up all main features of the System. Login is not required in order to reach this page. On the top, a fixed toolbar provides a comfy interface to navigate services offered by iNeed. On the bottom, there are buttons to check social platforms, get iNeed team contacts and read iNeed policy.


Login

A login form will be displayed to the user whenever they attempts to access a service which requires login. Login can be performed either via iNeed account or via a social media account, e.g. Facebook, Google, ...


MakeAnOffer

This is the first step in order to publish an Offer Ad. The user at this moment can set title, scope, type, and description of the Ad. The user can check regulations via a link on the bottom, to be sure that their ad is suitable with iNeed policy.


ViewFlow

Here all the ads published by the users are displayed, updated to the latest changes. The user can browse the ads by scrolling the table, or by querying a filtered search. On the bottom right, the user can access posting rules in order to properly flag non compliant ads. On the bottom left, a legend helps the user to disntinguish Offer Ads from Request Ads.


ViewMap

Here a map of the nearby location of the user is displayed. If a user publishes an ad, a pinpoint is put on the respective location on the map. On the left, a closeable toolbar offers several filters: by category, by distance and by key words.


ViewProfile

In this page all personal informations, which the user gave their agreement to show, are gathered. If the user flagged theirself as an Unemployed User, Company Talent Scouts can use this page to decide if that user is suitable for recruitment. This page can be accessed either by the owner of the profile in read-write mode, allowing they to customize it, or by other users in read-only.
